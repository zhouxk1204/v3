# AI写作风格检测与人类化改写工具

## 功能特性

### 1. AI写作风格分析
- 智能检测文本中的AI写作痕迹
- **AI写作指数**：0-100分的整体评分，直观展示AI化程度
- **8项量化指标**：全方位分析写作特征
- 饼图可视化展示AI特征与人类写作占比
- 提供具体改进建议

### 2. 量化指标体系
提供8个固定的量化指标，每个指标都有明确的评分范围和理想区间：

| 指标 | 范围 | 理想区间 | 说明 |
|------|------|----------|------|
| **用词正式度** | 0-100 | 30-60 | 词汇的正式程度，过高表示过于书面化 |
| **句式复杂度** | 0-100 | 40-70 | 句子结构的复杂程度，过高表示过于工整 |
| **逻辑结构化** | 0-100 | 50-80 | 内容组织的条理性，过高表示过于程式化 |
| **情感表达度** | 0-100 | 60-100 | 情感色彩的丰富程度，越高越好 |
| **个人风格化** | 0-100 | 60-100 | 个性化表达的程度，越高越好 |
| **自然过渡度** | 0-100 | 70-100 | 段落间过渡的自然程度，越高越好 |
| **词汇多样性** | 0-100 | 60-100 | 用词的丰富程度，越高越好 |
| **模板化程度** | 0-100 | 0-30 | 使用模板化表达的程度，越低越好 |

每个指标都会显示：
- 当前分数
- 可视化进度条（带理想范围标记）
- 颜色编码（绿色=理想，黄色=可接受，红色=需改进）
- 详细描述和理想范围

### 3. 场景化改写
支持6种改写场景：
- **论文**：学术严谨，专业表达
- **通用降AI**：去除AI痕迹，自然流畅
- **自媒体**：生动活泼，情感共鸣
- **职场**：专业得体，简洁明了
- **营销**：吸引力强，激发购买
- **学生作文**：朴实真挚，符合年龄

### 4. 实时流式输出
- 采用SSE流式传输
- 实时显示分析和改写结果
- 量化指标在前500字符内提取并展示
- 提升用户体验

## 技术实现

### 量化指标提取
```typescript
interface AnalysisMetrics {
  vocabularyFormality: number;      // 用词正式度
  sentenceComplexity: number;       // 句式复杂度
  logicalStructure: number;         // 逻辑结构化
  emotionalExpression: number;      // 情感表达度
  personalStyle: number;            // 个人风格化
  naturalTransition: number;        // 自然过渡度
  vocabularyDiversity: number;      // 词汇多样性
  templatePattern: number;          // 模板化程度
}
```

### 指标可视化
- 使用ECharts绘制饼图展示AI指数
- 进度条展示各项指标分数
- 理想范围标记（蓝色半透明区域）
- 颜色编码反馈（绿/黄/红）

## 使用方法

1. 在文本框中输入需要分析或改写的文本
2. 选择改写场景（默认为"通用降AI"）
3. 点击"分析AI写作风格"查看：
   - AI写作指数饼图
   - 8项量化指标详情
   - 详细文字分析
4. 点击"按场景一键改写"获取改写后的文本
5. 点击"复制"按钮复制改写结果

## 页面访问

开发环境：`http://localhost:5173/aiStyleAnalyzer`

## 注意事项

- 需要配置正确的API地址（.env.development中的APP_API_BASE_URL）
- 后端需要支持DeepSeek API的/chat/ask接口
- 建议输入文本长度在100-2000字之间效果最佳
- 量化指标会在流式输出的前500字符内提取并展示
