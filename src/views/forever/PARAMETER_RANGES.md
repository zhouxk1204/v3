# 参数范围说明

## 扭曲效果参数范围

### 网格密度 (Grid)
- **范围**: 3 - 50
- **默认**: 15
- **说明**: 控制扭曲网格的精细度，数值越大网格越密，扭曲效果越精细
- **建议**: 
  - 3-10: 粗糙效果，适合大幅度扭曲
  - 15-25: 平衡效果，适合大多数场景
  - 30-50: 精细效果，适合细腻的扭曲

### 鼠标影响半径 (Mouse Radius)
- **范围**: 0.05 - 1.0
- **默认**: 0.25
- **说明**: 鼠标影响的范围大小，数值越大影响范围越广
- **建议**:
  - 0.05-0.2: 小范围精确控制
  - 0.2-0.5: 中等范围，平衡效果
  - 0.5-1.0: 大范围影响，适合整体扭曲

### 扭曲强度 (Strength)
- **范围**: 0.01 - 1.0
- **默认**: 0.15
- **说明**: 扭曲效果的强度，数值越大扭曲越明显
- **建议**:
  - 0.01-0.1: 微妙效果
  - 0.1-0.3: 中等强度，适合文字
  - 0.3-1.0: 强烈效果，适合艺术创作

### 恢复速度 (Relaxation)
- **范围**: 0.5 - 0.999
- **默认**: 0.9
- **说明**: 扭曲恢复到原状的速度，数值越接近1恢复越慢
- **建议**:
  - 0.5-0.8: 快速恢复，动态效果
  - 0.8-0.95: 中等恢复速度
  - 0.95-0.999: 慢速恢复，持久效果

## 模式差异

### 文字模式
- 显示背景颜色设置
- 显示文字颜色设置
- 背景色会影响最终效果

### 图片模式
- 不显示背景颜色设置
- 背景自动设为透明
- 图片本身提供视觉内容
- 扭曲效果直接作用于图片

## 性能建议

- 网格密度过高 (>40) 可能影响性能
- 在移动设备上建议使用较低的网格密度 (10-20)
- 强度过高可能导致视觉混乱，建议适度使用
