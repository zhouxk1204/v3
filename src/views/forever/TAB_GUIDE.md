# Tab 功能说明

## 功能概述

新增了 Tab 切换功能，将文字模式和图片模式分开管理，提供更清晰的用户界面。

## Tab 模式

### 文字模式
- **内容设置**: 文字输入框
- **文字颜色**: 颜色选择器和文本输入
- **共通设置**: 背景颜色、扭曲效果参数

### 图片模式  
- **内容设置**: 图片上传和URL输入
- **共通设置**: 背景颜色、扭曲效果参数
- **注意**: 图片模式下不显示文字颜色设置

## 智能切换

### 自动切换逻辑
- 应用包含图片的预设时，自动切换到图片模式
- 应用纯文字预设时，自动切换到文字模式

### 预设分类
**文字预设**:
- 默认、霓虹、经典、简约
- 火焰、海洋、森林、日落
- 星系、复古、赛博朋克、优雅

**图片预设**:
- 自然 (风景图片)
- 抽象 (抽象艺术图片)

## 使用流程

1. **选择模式**: 点击"文字模式"或"图片模式" Tab
2. **设置内容**: 
   - 文字模式: 输入文字内容和颜色
   - 图片模式: 上传图片或输入URL
3. **调整共通设置**: 背景颜色和扭曲效果参数
4. **应用预设**: 点击预设按钮快速应用样式

## 技术实现

- 使用 `computed` 属性根据当前 Tab 决定传给组件的内容
- 文字模式时 `imageUrl` 为空，图片模式时 `text` 为空
- 预设应用时自动切换到对应的 Tab
- 保持所有共通设置的响应式更新
