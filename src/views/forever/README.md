# Forever 页面 - Trae 组件演示

这是一个完整的 Trae 组件演示和配置页面。

## 功能特性

### 实时预览
- 顶部显示 Trae 组件的实时效果
- 所有参数修改都会立即反映在预览中

### 完整的参数控制

#### 内容设置
- **文字内容**: 输入要显示的文字
- **图片URL**: 输入图片地址（会覆盖文字显示）

#### 颜色设置
- **背景颜色**: 
  - 颜色选择器快速选择
  - 文本输入支持所有CSS颜色格式
  - 一键设置透明背景
- **文字颜色**: 
  - 颜色选择器和文本输入
  - 支持RGBA透明度

#### 扭曲效果参数
- **网格密度** (5-30): 控制扭曲效果的精细度
- **鼠标影响半径** (0.1-0.5): 鼠标影响范围
- **扭曲强度** (0.05-0.5): 扭曲效果的强度
- **恢复速度** (0.8-0.99): 扭曲恢复的速度

### 快速预设
- **默认**: 经典的透明背景白色文字
- **霓虹**: 深色背景配青色文字，强烈扭曲效果
- **经典**: 白色背景黑色文字，温和效果
- **简约**: 透明背景灰色文字，最小扭曲

### 配置管理
- **导出配置**: 将当前设置保存为JSON文件
- **导入配置**: 从JSON文件加载设置
- **重置**: 恢复到默认设置

## 使用方法

1. 在文字输入框中输入想要显示的内容
2. 使用颜色选择器或直接输入颜色值调整外观
3. 拖动滑块调整扭曲效果参数
4. 使用预设按钮快速应用不同风格
5. 可以导出满意的配置供后续使用

## 响应式设计

- 桌面端：上下分屏布局，预览区域较大
- 移动端：自适应布局，控制面板可滚动

## 技术实现

- Vue 3 Composition API
- 响应式数据绑定
- 实时参数同步
- 文件导入导出功能
- 现代CSS样式（渐变、毛玻璃效果）
